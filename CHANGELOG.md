# Changelog

All notable changes to `devletes/filament-pinnable-navigation` will be documented in this file.

## [1.0.0] - 2026-03-18

### Added

- Filament 5 panel plugin for pinnable grouped sidebar navigation.
- Sidebar pin toggles and page-header pin toggles for supported resource pages.
- LocalStorage persistence keyed by panel and authenticated user.
- Optional database-backed persistence for pinned items.
- Testbench workbench app for local package development and manual verification.
- Automated Pest test coverage for plugin registration, navigation behavior, and persistence modes.
- Package CSS and JavaScript registered through Filament's asset manager.
